# Quantum Untangled
<p align="center"><img src="https://github.com/Quantum-untangled/QuantumUntangledWebsite/blob/main/website/Quantum%20Untangled%20Logo.jpg" alt="Quantum Untangled Logo"/></p

This project aims to make quantum computing accessible to people of all ages. We want to bring the basics of quantum mechanics and quantum computing to a wider public. We intend to do this through a website that will work as a hub for all-things quantum. The website is still under development now, but expect a working version by the summer of 2021.

## Contributing
To contribute to the project, please open an issue describing the feature you want to implement or the bug that needs to be fixed. The issue will be reviewed and a pull request will be opened next.   

In order to work on the code in your local machine, you will need to first set up the Django framework and a virtual environment. Instructions for doing that can be found here: https://docs.djangoproject.com/en/3.1/intro/install/. After having this you need to follow the next steps:
- Clone the repository in your local machine
- Start a virtual environment in the folder you cloned the repository in
- Go into the `website` folder using the command line
- Open the file named `settings.py` and scroll down to the `DATABASES` variable, you'll find further instructions there
- Run `python manage.py runserver` from the commmand line
